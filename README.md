# LAN Chat Application - NodeTalk

## Project Overview

The LAN Chat Application - NodeTalk is a desktop-based communication system developed using Python. The application enables users connected to the same Local Area Network (LAN) to communicate with each other in real time through a user-friendly graphical interface.

This project was developed to gain practical experience in networking, socket programming, database management, and graphical user interface development using Python.

---

## Main Features

• User Registration and Login

• Real-Time Messaging

• File Sharing Between Users

• Active User List Display

• SQLite Database Integration

• Multi-threaded Server Handling

• User Join and Leave Notifications

## System Operation

1. The server is started and waits for incoming client connections.

2. Users can create a new account through the registration interface or log in using existing credentials.

3. After successful authentication, users enter the chat window and can participate in conversations.

4. Messages are transmitted through the server and delivered to all connected users in real time.

5. Files can be shared between connected users through the application.

6. The active user list is automatically updated whenever users join or leave the chat.
